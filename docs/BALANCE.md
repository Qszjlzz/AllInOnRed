## 1. 时间与经济

| 参数 | 范围/值 | 说明 |
|------|---------|------|
| 游戏天数 | 3 | 约 10–20 分钟 |
| 每日 AP | 3 | 赌博/工作/取出各耗 1 |
| 初始现金 | ¥800–1200 | 随机 |
| 账单总额 | ¥3000–5000 | 随机，HUD 显示进度 |
| 虚拟余额计债 | ×0.5 | 机器里的钱只算一半资产 |

## 2. 工作 QTE

| 参数 | 值 |
|------|-----|
| 每轮总点击 | 5 |

| 成功收入 | ¥400–700 |
| 失败收入 | ¥80–150 |
| 勤勉触发 | 15% 成功时 +1 勤勉 |

### 上瘾惩罚

| 上瘾层数 | 光标速度 | 收入倍率 |
|----------|----------|----------|
| 0 | 100% | 100% |
| 1 | 85% | 90% |
| 2 | 70% | 75% |
| ≥3 | 55% | 60% |

## 3. 赌博

| 参数 | 值 |
|------|-----|
| 首次按 | 免费（不占 AP） |
| 之后每次 | 1 AP |
| 单次下注 | ¥80–150（自动扣） |
| 三连按 | Day2 解锁，1 AP 转 3 次 |
| 存入机器 | 全额现金→虚拟，无 AP |
| 取出 | 1 AP，¥200–500 随机上限 |

### 上瘾获得概率

| 当前上瘾 | 每次赌博 +1 概率 |
|----------|------------------|
| 0 | 15% |
| 1 | 25% |
| 2+ | 40% |
| 三连按额外 | +15% |

上瘾 ≥3 触发强制再按事件（Day2+）。

## 5. 心情系统

| 状态 | 获得 | 效果 |
|------|------|------|
| 上瘾 | 赌博概率 | 工作减速/减收；≥3 强制赌 |
| 踏实 | 工作成功、回复家人 | 抵消 1 不安 |
| 不安 | 忽略家人、亏损 | ≥3 触发透支事件 |
| 勤勉 | 工作 15% | 抵消 1 上瘾；工作加速 |

## 6. 结局阈值

| 结局 ID | 条件 |
|---------|------|
| early | Day1 弹窗选「关掉了」 |
| early_family | Day1 同事弹窗选「关网页，回家陪家人」 |
| phone_dead | Day1 赌≥5 且未早期止损 |
| perfect（回家） | 最终不按 + 债务比 <30% + 赌博 ≤4 |
| awaken（幡然醒悟） | 最终不按 + 债务比 <70% |
| stop_loss（及时止损） | 最终不按 + 债务比 ≥70% |
| ruin（沉沦） | 最终【按】（首次通关） |
| memory（前世记忆） | 重玩 + 曾达坏结局 + 最终不按 |
| delusion（执迷不悟） | 重玩 + 曾达坏结局 + 最终【按】 |

债务比 = 待还 / 账单总额。重玩记录键：`localStorage.biean_last_ending`。

---

## 7. 设计意图

- **Day1 偏赢**：降低门槛，制造「可以试试」的错觉  
- **Day3 偏亏**：账单与家庭事件叠加，压力峰值  
- **虚拟余额**：提现耗 AP，模拟「钱变数字」的心理摩擦  
- **不上瘾也能玩通**：工作+节制赌博可走「回家」线
