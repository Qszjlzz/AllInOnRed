# 数值策划 Agent Brief — 别按那个键

## 设计目标

1. **工作**：慢、稳、占 1 AP，Day3 前可维持收支  
2. **赌博**：快、波动大，期望值为负（长期必亏）  
3. **虚拟余额**：存入容易，取出占 AP，制造「钱只是数字」感  
4. **Demo 10–15 分钟**可体验完整弧线

---

## 初始值（Demo）

| 变量 | 范围/值 | 说明 |
|------|---------|------|
| 起始现金 | 800–1200（随机） | `rand(800,1200)` |
| 账单总额 | 3000–5000 | 3 天内需心理压力，非必须还清 |
| 起始虚拟余额 | 0 | — |
| 每日 AP | 3 | 固定 |

---

## 工作 minigame

| 参数 | 值 |
|------|-----|
| 每 day 次数 | 1（第二次起效率 -50% 或禁止，Demo 禁止） |
| 成功收入 | `rand(400, 700)` |
| 失败收入 | `rand(80, 150)` 或 0（Demo 给低保） |
| 成功判定 | QTE 命中 ≥3/5 |
| AP 消耗 | 1 |
| 踏实 | 成功 +1 |

---

## 赌博 — 单次按

**消耗**：1 AP（Day1 首次免费引导除外）

### 转盘权重（8 格）

| 结果 | 权重 | 金额效果 |
|------|------|----------|
| 小赢 | 25% | +`rand(50,150)` |
| 中赢 | 10% | +`rand(150,300)` |
| 大赢 | 3% | +`rand(300,500)` |
| 小亏 | 30% | -`rand(50,120)` |
| 中亏 | 20% | -`rand(120,250)` |
| 翻倍 | 5% | 当前机器余额 ×2（上限 5000） |
| 清空 | 7% | 机器余额归零 |
| 持平 | 0% | 0 |

**期望**：约 -5% ~ -15% 每次（相对本次赌注基线 100）

**赌注基线**：优先扣机器余额，不足扣现金 `rand(80,150)`

### 上瘾 proc

| 上瘾层数 | 赌博后 +上瘾 概率 |
|----------|-------------------|
| 0 | 15% |
| 1 | 25% |
| 2+ | 40% |

---

## 赌博 — 三连按（Day2+）

- **消耗**：1 AP  
- **效果**：连续转 3 次，间隔 0.5s  
- **解锁**：Day2 且 addiction≥1 或 Day1 未 early stop  
- **额外**：每次 +15% 上瘾概率

---

## 虚拟余额存取

| 操作 | AP | 规则 |
|------|-----|------|
| 存入 | 0 | 任意现金 → 机器，上限 99999 |
| 取出 | 1 | 最多取 `min(余额, rand(200,500))` 或全额（Demo 全额需 1 AP） |

**心理设计**：存入无成本，取出有摩擦。

---

## Mood 阈值

| 状态 | 触发 | 效果 |
|------|------|------|
| 上瘾 ≥3 | 自动 | Day3 前触发「强制再按一次」事件（占 0 AP） |
| 踏实 ≥2 | 被动 | 抵消不安 1 层 |
| 不安 ≥3 | 自动 | 随机：扣 `rand(100,300)` 或跳过 family 回复奖励 |

---

## 账单与债务

```
显示债务 = max(0, 账单总额 - 已还金额)
已还金额 = 累计（现金还款 + 自动工资扣款 stretch）
```

**Demo 简化**：  
- 不强制还款 minigame，仅显示进度条  
- `billPressure = 债务 / 账单总额` 用于结局权重

**自动扣款（可选 stretch）**：Day 结束扣 `rand(100,200)` 若现金足够

---

## 健康 / 透支

- **Demo 不做卖血按钮**  
- 当 `debt > 15000` 或 ` addiction≥4`：事件「透支健康」→ 现金 +`rand(500,800)`，不安 +2，文案惩罚

---

## 结局判定矩阵

| 结局 ID | 条件 |
|---------|------|
| `early` | Day1 选择「关掉了」 |
| `perfect` | final 选不按 AND debt<30% AND 赌博次数≤3 |
| `awaken` | final 不按 AND debt<70% |
| `stop_loss` | final 不按 AND debt≥70% |
| `ruin` | final 按 OR debt≥100% OR 虚拟余额>现金×2 且按 |

**债务计算**：
```
debt = billTotal - cash - virtualBalance*0.5
```
（虚拟余额算半额「心理资产」）
