# 别按那个键 — Demo 范围计划

> 融合来源：v8.1 叙事 PDF、桌面化实现方向、策划案改（密教模拟器卡牌简化版）

---

## 1. 核心概念

**一句话**：深夜加班的上班族，在电脑桌面上收到朋友发来的 `别按那个键.html`，在「工作赚钱」与「赌博快钱」之间消耗每日行动点，最终面对【按】/【不按】的抉择。

**呈现层**：Windows 风格桌面模拟器（壁纸、任务栏、通知、可拖拽窗口）  
**机制层**：简化的卡牌/行动点系统（非完整 Cultist Simulator）  
**叙事层**：v8.1 五幕结构压缩为 Demo 的 3 天

---

## 2. Demo 时间结构

| 原设计 | Demo 调整 |
|--------|-----------|
| 60 秒 / 周期 | **1 天 = 1 周期** |
| 每周期 1 行动格 | **每 day 3 行动点（AP）** |
| 整局 10+ 周期 | **Demo 共 3 天**（约 10–15 分钟） |

### 每日流程
1. 早晨/班前：可能收到通知或家人消息（被动事件，不占 AP）
2. 玩家消耗 AP 打开应用：聊天 / 工作 / 赌博 / 备忘录
3. AP 用尽或玩家主动「结束今天」→ 进入下一天
4. 第 3 天晚间触发账单压力 → 女儿画画 → 自我对话 → 最终抉择

---

## 3. 桌面应用映射

| 窗口 | 叙事功能 | 机制功能 |
|------|----------|----------|
| **聊天**（朋友） | Day1 发送 `别按那个键.html` | 剧情引导、文件链接 |
| **别按那个键.html** | 赌博主界面 | 按键 + 转盘；虚拟余额存储 |
| **工作软件** | 加班赚钱 | QTE 修机式小游戏，成功得现金 |
| **家庭群聊** | 妻子/女儿消息 | Day2 关心、Day3 女儿画作事件 |
| **备忘录** | 自我对话 | 玩家输入文本，Day3  climax 回放 |
| **账单/邮件**（可选） | 还款压力 | 显示账单进度条 |

---

## 4. 故事节拍 → 桌面事件

### Day 1（好奇 → 初瘾）
| 节拍 | 桌面呈现 |
|------|----------|
| 朋友发来文件 | 聊天窗口通知 + 可点击链接 |
| 第一次按，小赢 | 赌博窗：转盘 → 随机 +50~200 |
| 再按，输掉 | 转盘 → 随机 -80~150 |
| 早期止损选择 | 弹窗：「今天就到这？」→ 结局 A（及时收手）/ 继续 |
| 上瘾 mood +1 | 状态栏显示「上瘾」计数 |

### Day 2（习惯 → 对比）
| 节拍 | 桌面呈现 |
|------|----------|
| 工作可用 | 任务栏工作图标高亮 |
| 家人消息 | 家庭群：「今晚回来吃饭吗？」 |
| 赌博进阶 | 解锁「三连按」（若 Day1 继续赌博） |
| 存钱进机器 | 赌博窗「存入数字」按钮，现金 → 虚拟余额 |
| 不安 mood | 忽略家人 → 不安 +1 |

### Day 3（账单 → 失控 → 抉择）
| 节拍 | 桌面呈现 |
|------|----------|
| 账单提醒 | 系统通知 + 邮件窗口 |
| 女儿画作 | 家庭群图片占位 + 「爸爸别熬夜」 |
| 自我对话 | 备忘录强制输入：「你想对自己说什么？」 |
| 最终抉择 | 赌博窗全屏 overlay：【按】/【不按】 |
| 回放 | 显示玩家 Day3 备忘录原文 |
| 结局 | 按债务 + 选择分支 |

---

## 5. 心情系统（Demo 简化）

| 状态 | 获取 | 效果 |
|------|------|------|
| **上瘾** | 赌博后概率获得 | ≥3 触发强制十连按事件 |
| **踏实** | 工作成功 / 回复家人 | 抵消 1 不安 |
| **不安** | 忽略家人 / 大额亏损 | ≥3 触发负面事件（Demo：扣钱或 mood 惩罚） |

> Demo 不做完整卡牌拖拽，以顶部状态栏数字 + 事件文案呈现。

---

## 6. 赌博机 Demo 范围

- **Day 1**：单次按 + 转盘（翻倍 / 清空 / ±随机金额）
- **Day 2**：三连按（若已上瘾路径）
- **Day 3**：十连按（若上瘾 ≥3，或 stretch）
- **虚拟余额**：可将现金存入机器，提现需消耗 AP
- **不做**：老虎机、刮刮乐、卖血按键（用「透支健康」文案隐喻，Demo 仅扣 mood）

## 9. 成功标准

1. 新玩家无需说明，10 分钟内可打完 Demo
2. 能清晰感受「工作慢但稳 / 赌博快但险」对比
3. 「存钱进机器」产生数字依赖感
4. 备忘录输入在结局被回放，情感落地
5. 团队可按 TEAMS_ROLES 分工并行推进

## 10. 与完整版差异

| 项目 | Demo | 完整版 |
|------|------|--------|
| 天数 | 3 天 | 7–10 天 |
| AP | 3/天 | 3/天 + 事件占格 |
| 卡牌 | 计数器 | 可视化卡牌 + 牌桌 |
| 赌博 | 按钮+转盘 | + 老虎机、刮刮乐 |
| 结局 | 4+ | 6+ 含隐藏 |
| 同事小王线 | 省略 | Day4+ 打断事件 |
