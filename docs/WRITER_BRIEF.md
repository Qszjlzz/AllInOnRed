# 文案 Agent Brief — 别按那个键

## 任务概述

为 HTML 桌面 Demo 撰写**全部玩家可见中文文案**。基调参考 v8.1 PDF《别按那个键》：深夜加班、好奇、温水煮青蛙、账单压顶、家庭撕裂、最终自我对话与按键抉择。

---

## 语气指南

| 维度 | 要求 |
|------|------|
| 视角 | 第二人称隐性（「你」），UI 用系统口吻 |
| 朋友 | 调侃、轻描淡写危险（「试试呗」） |
| 妻子小雅 | 关心但渐显失望，短句、微信体 |
| 女儿朵朵 | 童真、拼音错误可少量使用 |
| 内心 | Day1 好奇 → Day2 自我安慰 → Day3 撕裂 |
| 禁止 | 说教式反赌口号、血腥卖器官描写（用「透支健康」） |
| 长度 | 单条聊天 ≤40 字；旁白 ≤80 字 |

|------|------|----------|

## 需交付文案清单

### A. 全局 UI
- [ ] 游戏标题、开始界面说明（2–3 句）
- [ ] 任务栏应用名：聊天、工作、赌博网页、家庭群、备忘录
- [ ] AP 显示：「今日行动：{n}/3」
- [ ] 金钱：「现金 ¥{n}」「机器余额 ¥{n}」
- [ ] 账单：「待还 ¥{n} / 已还 ¥{n}」
- [ ] Mood 标签：上瘾、踏实、不安
- [ ] 按钮：结束今天、存入机器、取出、按、不按、再试一次

### B. Day 1 事件

**event.d1.friend_open** — 聊天开场  
> 示例方向：阿凯深夜丢链接，「加班呢？给你个东西解闷，别按啊——骗你的」

**event.d1.file_received** — 文件卡片  
> 文件名：`别按那个键.html`  
> 描述：「一个简陋的网页，中间只有一个按钮。」

**event.d1.first_win** — 首次赌博后  
> 示例：「屏幕跳出一串数字。心跳快了一点。」

**event.d1.first_loss** — 第二次赌后  
> 示例：「数字又暗下去。你告诉自己，只是运气不好。」

**event.d1.early_stop** — 早期止损弹窗  
> 标题：「今晚到此为止？」  
> 正文：「你还没欠什么。现在关掉，还来得及。」  
> 选项：【关掉了】/【再按一次】

**ending.early_stop** — 早期结局  
> 标题：「及时收手」  
> 正文：需写 3–4 句，温和落地

### C. Day 2 事件

**event.d2.work_available** — 工作通知  
**event.d2.family_ping** — 小雅：「今晚回来吃饭吗？朵朵想你了。」  
**event.d2.family_reply_good** — 玩家可选回复（踏实 +1）  
**event.d2.family_reply_ignore** — 忽略后果（不安 +1）  
**event.d2.triple_unlock** — 解锁三连按提示  
**event.d2.deposit_hint** — 首次可「存入机器」时的系统说明（心理依赖）

### D. Day 3 事件

**event.d3.bill_notice** — 账单通知（含具体感压力，不写绝对数字，用「这个月的」）  
**event.d3.daughter_drawing** — 朵朵发图 + 「爸爸别熬夜好不好」  
**event.d3.memo_prompt** — 备忘录：「你想对今晚的自己说什么？（会记下来的）」  
**event.d3.climax_replay_prefix** — 最终抉择前：「那天夜里，你写下——」

### E. 最终抉择

**event.final.choice_title** — 「最后一次，你把手放在鼠标上。」  
**btn.press** — 【按】  
**btn.not_press** — 【不按】

### F. 结局（至少 4 条）

| ID | 条件概要 | 标题建议 |
|----|----------|----------|
| ending.perfect | 几乎不赌 + 不按 + 低债务 | 完美收束 |
| ending.awaken | 中债务 + 不按 | 幡然醒悟 |
| ending.stop_loss | 高债务 + 不按 | 及时止损 |
| ending.ruin | 极高债务或按 | 沉沦 |
| ending.early | Day1 止损 | 今晚关闭 |

每条：标题 + 80–120 字旁白。

## 输出格式建议

```json
{
  "event.d1.friend_open": "阿凯：还在公司？发个东西给你……",
  "ending.early_stop.title": "及时收手",
  "ending.early_stop.body": "你关掉了网页……"
}
```

文件路径建议：`data/strings.json`  
程序 fallback：内联中文（当前 Demo 已含初稿，文案负责润色替换）

## 验收标准

1. 无错别字、无英文夹杂（除文件名 html）
2. Day1→3 情感曲线可读
3. 早期止损与最终抉择文案不重复
4. 女儿线不煽情过度
5. 所有 `{变量}` 与程序占位一致
